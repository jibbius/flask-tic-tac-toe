About

This is my final project for General Assembly's Python Programming course.
As part of this project, I am seeking to;

  • Test my competencies with Flask and Persisting Data
  • Showcase an implementation of the Strategy Pattern
    • I have used this design pattern, in the context of different computer-opponent difficulties

Features

  • Ability to seed initial list of Players from CSV file.
  • Persistence layer that leverages SQLAlchemy ORM and a SQLite database.
  • Different difficulty levels for computer-controlled players
  • Web UI for all basic game functions
    • Ability to create a game of Tic Tac Toe.
    • Ability to assign players to a game.
    • Ability for HUMAN players to submit moves, to progress a game.
    • Ability for COMPUTER players to submit moves, to progress a game.
    • Ability for a game to determine its winner.
  • Full JSON API catalogue, to; view game data, add/update players, and play tic tac toe.
  • HTML + JS forms to test APIs.
